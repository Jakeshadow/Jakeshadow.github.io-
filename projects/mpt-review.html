What Is MoneyPrinterTurbo?

MoneyPrinterTurbo is an open-source AI tool that automatically generates videos from a single sentence. You give it a topic (e.g. "Top 5 facts about black holes") and it produces a complete video — script, voiceover, visuals, subtitles, and background music — without you touching a timeline.

It was built by a Chinese developer (harry0703) and gained massive traction on GitHub for being the first truly one-click AI video pipeline that runs locally.

Is It Free?

Real Setup Guide (Took Me 1.5 Hours. You? 10 Minutes.)

Every tutorial says "quick and easy." Here's what actually works, including the parts everyone else skips.

  1. Pick the RIGHT version
    The latest release (v1.2.9) is API-only — no Web UI. Beginners must use the old version:
    git clone --branch v1.1.0 --depth 1 https://github.com/harry0703/MoneyPrinterTurbo.git
    cd MoneyPrinterTurbo
    VPN ON for git clone. The connection resets constantly without it. Use --depth 1 or it'll fail mid-transfer.
  2. Install dependencies — with the right flags
    Python 3.10+ required. VPN OFF. Default pip is too slow and WILL timeout. Use Aliyun mirror:
    pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/
    If you get "moviepy.editor not found": pip install moviepy==1.0.3 — v2.0.0 broke the API MPT v1.1.0 depends on.
  3. Configure in 30 seconds
    Copy config.example.toml → config.toml. Change ONE line:
    llm_provider = "g4f"
    That's it. g4f = free AI. Edge TTS = free voiceover. No API keys. No Azure registration. No credit card.
  4. Launch the BACKEND first
    Open PowerShell #1:
    cd D:\Projects\MoneyPrinterTurbo
    python main.py
    Wait for "start server, docs: http://127.0.0.1:8080/docs". Do NOT close this window.
  5. Launch the FRONTEND
    Open PowerShell #2:
    cd D:\Projects\MoneyPrinterTurbo
    streamlit run .\webui\Main.py
    Browser opens at http://localhost:8501. Type a topic, click Generate. VPN ON if it hangs at "downloading materials."
Why so many steps? MPT is a community project with rapid version churn. The v1.2.x rewrite dropped the GUI. The pip dependencies are pinned to incompatible versions. No one documented this — until now. This guide turns a 1.5-hour debugging session into a 10-minute copy-paste job.
